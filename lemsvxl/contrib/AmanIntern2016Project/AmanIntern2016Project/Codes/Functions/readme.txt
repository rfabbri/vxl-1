This file describes the various functions contained in this folder

1)EXTRACT_FEATURES - This function extract features from an input image, meaning SIFT, maxima or minima features as desired.
2)EXTRACT_POINTS - This function takes two inputs and returns as output a matrix of points that we want to fix, ie, maxima, minima, saddles, watershed or watercourse
3)EXTRACT_FEATURES - This function takes the descriptors of two images which have been extracted beforehand, and returns the matching results.
4)IMG_RECONSTRUCTION - This function takes three inputs and returns the reconstruced image as output. Reconstruction is based on the solution of the Heat Equation.
